numbers, matched symbols, commands run), not narrative summaries. Prefer broad
search for filename/extension shape, targeted grep for symbols or keywords, and
file reads only when excerpts cannot resolve the question.

import Copilot-only lifecycle, mediator chains, council, vote, or consensus
semantics. Report what you searched, what you found, and what remains
unverified so the parent can decide whether to widen scope.
